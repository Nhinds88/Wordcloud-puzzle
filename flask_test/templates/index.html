Welcome to Cluzzle!
The goal of the game is to guess the single word the word cloud above is created from. The text in the cloud is made up from quotes about the word we trying to guess and the cloud is in the shape of a random image pulled based off the word as well.
You will get four guesses and after each quess the image will change and the quoted text in the cloud will change as well.
If you feel the need there will be several hints you can use.

1. Wiki Button: provides a small portion of the wiki entry for the word.
2. Letter Count Button: provides the number of letters in the word.
3. Synonym Button: will provide a synonym for the word.
